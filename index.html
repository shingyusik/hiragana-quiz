히라가나 암기 퀴즈

테스트를 시작하면 히라가나가 랜덤으로 하나씩 나타납니다. 키보드로 발음을 입력하세요. 기본은 46자이며, 탁음은 버튼으로 켰을 때만 함께 출제됩니다. 한국어와 영어 표기 둘 다 정답으로 인정합니다.

현재 문제
?

시작 버튼을 누르면 문제가 표시됩니다.

현재 문제풀: 기본 46자

정답 입력
퀴즈를 시작하면 여기에 결과가 표시됩니다.
진행 상태
남은 문제 46
정답 수 0
연속 정답 0
진행률 0 / 46
입력 예시
  • し는 shi, si, 시 모두 인정
  • ち는 chi, ti, 치 모두 인정
  • つ는 tsu, tu, 츠 모두 인정
  • 입력은 대소문자와 공백을 무시합니다.
